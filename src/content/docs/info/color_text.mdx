The Color Text component is used to inline a color code, along with a small box to the left of the text displaying the color.

It works based on the css color code, so any valid css color code can be used.



# Usage
```js
<ColorText color="#79FC92" />
<ColorText color="red" />
<ColorText color="rgb(0, 0, 255)" />
```
Which will render as:
<ColorText color="#79FC92" />
<ColorText color="red" />
<ColorText color="rgb(0, 0, 255)" />

# Props
| Name  | Description                    | Type   | Default  |
|-------|--------------------------------|--------|----------|
| color | The css  color code to display | String | Required |
