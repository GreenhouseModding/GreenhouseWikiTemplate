# Minecraft Icon List

- food <MinecraftIcon icon="food" />
- heart <MinecraftIcon icon="heart" />

# Usage
```js
<MinecraftIcon icon="food" size="2em"/>
```
Which will render as <MinecraftIcon icon="food" size="2em"/>

# Props
| Name  | Description                                   | Type                                              | Default   |
|-------|-----------------------------------------------|---------------------------------------------------|-----------|
| icon  | The name of the icon to display               | Enum: "food", "heart"                             | Required  |
| size  | The size of the icon to display               | String                                            | Optional  |
